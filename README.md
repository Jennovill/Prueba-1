# Observatorio de Extorsiones en Guayaquil
<img width="1892" height="917" alt="image" src="https://github.com/user-attachments/assets/55d6e8f0-03de-4881-87be-3811197c10af" />

Plataforma web desarrollada para visualizar, analizar y comprender el fenómeno de las extorsiones en Guayaquil mediante información estructurada, análisis territorial y modelos predictivos.

---

## Objetivo

El proyecto tiene como finalidad:

- Visualizar datos históricos de extorsiones (2021–2024).
- Analizar patrones geográficos.
- Explorar tendencias temporales.
- Generar predicciones estadísticas.
- Facilitar el acceso a información clara y verificable.

## Contexto Académico

Proyecto desarrollado en el marco académico de la  
**Facultad de Ciencias Naturales y Matemáticas – ESPOL**,  
en colaboración con el **Centro de Estudios e Investigaciones Estadísticas**.

El sistema busca aportar una base cuantitativa y analítica para el estudio del fenómeno de las extorsiones en el contexto local.
